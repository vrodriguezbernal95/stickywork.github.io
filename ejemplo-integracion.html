Ejemplo de Integración

Widget de StickyWork en una página externa

Sobre este ejemplo

Esta página demuestra cómo integrar el widget de StickyWork en cualquier sitio web. El widget está completamente funcional y conectado a una base de datos real. Puedes usar este código como base para tu propia implementación.

Código de integración

Solo necesitas agregar estas 3 líneas de código a tu página HTML:

<div id="stickywork-widget"></div>

<script src="http://localhost:3000/widget/stickywork-widget.js"></script>

<script>
  StickyWork.init({
    businessId: 1,
    apiUrl: 'http://localhost:3000',
    primaryColor: '#3b82f6',
    language: 'es'
  });
</script>
